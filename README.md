# Sequence Stitch Player

一个面向 Windows 11 的极简图片序列拼接播放器，用来快速检查多个 C4D 预演序列帧之间的衔接是否自然。

## 安装方法

```bat
cd sequence_stitch_player
py -3.12 -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## 运行方法

```bat
python main.py
```

或者双击：

```bat
run.bat
```

`run.bat` 需要先按安装步骤创建好 `.venv` 并安装依赖。

## 支持格式

- `.jpg`
- `.jpeg`
- `.png`

大小写不敏感。每个文件夹代表一个 clip，软件会按文件夹顺序将 A、B、C 三段拼接成一条全局时间线。

## 显示尺寸

`Display Size` 可以选择：

- `Fit Window`
- `1280 x 720`
- `1920 x 1080`

固定显示尺寸不会裁切或拉伸图片，仍然会保持宽高比，并以黑色背景填充空白区域。

## 素材优化

选择 `1280 x 720` 或 `1920 x 1080` 后，可以点击 `Optimize Folder...` 批量降低某个文件夹内的图片分辨率。

注意：这个功能会覆盖原始图片。请先备份素材。`Fit Window` 模式下该按钮会禁用，避免误用当前窗口尺寸覆盖素材。

优化规则：

- 只处理 `.jpg` / `.jpeg` / `.png`
- 只缩小，不放大
- 保持宽高比
- 处理完成后自动 Reload

## 文件变更提示

暂停播放时，程序会每 15 秒轻量检查一次已选择文件夹的文件列表、修改时间和文件大小。

窗口失去焦点超过 2 分钟后会进入 `Idle`，暂停检查以节省资源。重新激活窗口后会恢复检查。

检测到变更时只显示提示，不会自动 Reload，避免读取正在写入的半成品文件。

## 快捷键

| 快捷键 | 功能 |
| --- | --- |
| `Space` | 播放 / 暂停 |
| `Left` | 后退 1 帧 |
| `Right` | 前进 1 帧 |
| `Shift + Left` | 后退 10 帧 |
| `Shift + Right` | 前进 10 帧 |
| `Home` | 跳到第一帧 |
| `End` | 跳到最后一帧 |
| `R` | 重新扫描所有文件夹并刷新序列 |
| `L` | 开关 Loop |

程序内也可以点击 `Shortcuts` 查看快捷键说明。

## 已知限制

- 第一版不支持 `mp4` / `mov`
- 第一版不支持音频
- 第一版不支持自动热刷新
- 第一版不支持 `exr` / `tif` / `tiff`
- 目前只提供 A / B / C 三个固定 clip 槽位
- 超高分辨率 PNG 序列的播放流畅度依赖机器性能
